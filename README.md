# FinFlex AI - Modern Financial Intelligence

Modular finance platform with real-time simulations and AI insights.

## Architecture
- **Frontend**: React + Vite + Tailwind CSS (Stripe Design System)
- **Backend**: Express (Modular API Routes)
- **Database**: Firebase Firestore (Enterprise Tier)
- **AI**: Gemini 3.1 Flash (Google Gen AI)
- **Infrastructure**: CI/CD (GitHub Actions), Docker ready.

## Deployment
1. Copy `.env.example` to `.env`
2. Run `npm install`
3. Run `npm run dev`
